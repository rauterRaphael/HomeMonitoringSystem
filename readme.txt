##### sensor_node ##### 

Upload sketch via Waspmote IDE and turn node on with hibernate enabled

##### dashboard && gateway ##### 

- Create new python virtualenv: "python -m venv hms"
- Activate environment: "./hms/Scripts/activate"
- Install requirements: "pip install -r requirements.txt"

### dashboard

- Open new shell
- Activate environment: "./hms/Scripts/activate"
- Start django with "python manage.py runserver" in shell

### gateway

- Open new shell
- Activate environment: "./hms/Scripts/activate"
- Start gateway application with "python runGateway.py" in shell
